Проект: Классификация изображений отпечатков пальцев при помощи сверточной нейросети

Датасет - https://www.kaggle.com/datasets/ruizgara/socofing

Описание:
Проект включает в себя создание и обучение сверточной нейронной сети с использованием библиотеки TensorFlow. Нейросеть предназначена для классификации изображений отпечатков пальцев (какой палец какой руки на изображении) из набора данных, скачанного с ресурса https://www.kaggle.com/datasets/ruizgara/socofing.
